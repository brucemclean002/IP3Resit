Where to Source Data

Finding reliable and interesting data sources is crucial for effective data visualization. Some popular sources for data include:

  • CDC COVID-19 Data
  • OpenWeatherMap
  • Data.gov
  • Data.World

Data Content and Structure

Understanding the structure and content of your data is important for visualization. Data is often structured in JSON or CSV formats, and typical elements include:

  • Attributes: Characteristics or properties of the data (e.g., temperature, date, location).
  • Records: Individual entries in the dataset (e.g., a day's weather data).

API Access

APIs (Application Programming Interfaces) allow you to access data from external sources programmatically. To use an API, you typically need to:

  1. Register for an API key if required.
  2. Understand the API endpoints and parameters.
  3. Make HTTP requests to the API.

Example API Call:


// Fetch COVID-19 data
fetch('https://disease.sh/v3/covid-19/all')
    .then(response => response.json())
    .then(data => console.log(data));
